# FlowableDoc

### 前言

目前这部分内容只是对flowable的一个入门教程，是在官方文档的基础上加上个人理解和代码示例形成的，以后有时间深入研究的话会继续更新在此仓库下。

并且由于flowable对于BPMN模型的介绍很多，这一部分可能需要读者自身去阅读官方文档加深理解，这个教程对于这部分的描述可能会不那么完全。
